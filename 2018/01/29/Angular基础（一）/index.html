Angular基础一

一些基础用法

Quick Start

@Component

@Component({
    selector: 'app-hello',   // 用于模版中的标签
    templateUrl: './hello.component.html',  // 模版
    styleUrls: ['./hello.component.css'] // 模版样式
})

*ngFor

*ngFor : 在一个列表上进行迭代，为列表的每一个条目生成一个新的标签。
<li *ngFor="let name of names">{{name}}</li>

@Input

@Input 可以将数据传输给自组件，能够接受输入。
@Input() name: string;
<app-hello *ngFor="let myname of names" [name] = myname></app-hello>

@NgModule

在创建组件时必须把组件放入@NgModule的declarations中。
@NgModule({
declarations: [
  AppComponent,
  HelloComponent
]    
})
