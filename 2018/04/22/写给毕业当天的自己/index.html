现在是北京时间2018年4月22日晚上10点10分,距离我毕业(6月20左右)只有两个月,在这两个月中我想做:

一、4月底完成毕设论文初版

初稿的要求: 对整个毕设有清晰的脉络,整体思路,细节都需要呈现.

二、5月15号完成毕设

经过指导老师确认,论文,毕设都准备齐全.

三、补习基础知识

java并发基础

对java的核心功能掌握,尤其是多线程

高并发分布式系统基础知识

重点放到分布式存储和消息中间件
tomcat 集群,nginx,MQ

(1)实战Java高并发程序设计[葛一敏]

(2)java 编程思想(4th)

(3)Spring_Microservices

Angular 5 整体思想

(1)Apress.Pro.Angular
能够自己设计构建系统,自定义可扩展的组件.

四、数据库、算法、设计模式

平时多积累,多做题

Python

(1)Python语法
(2)Python爬虫

Oracle & MySql 数据库

(1)常用sql语句
(2)函数,视图,存储过程
(3)优化及高级特性

五、交流

1.多和身边的人交流，有问题多问，越是感觉不好意思的问题越是要厚着脸皮问。
2.对一些即将要发生的，现在可以处理的事情，一定不要拖着，需要立即处理。
3.出现问题或者感觉不对，今早提出，寻求帮助。
4.每周和一些朋友打电话，视频聊聊天。

六、表现

重要习惯(坚持每天都做)

早晨听英文，看笔记
晚上写博客，笔记

每周都要给自己定目标:工作目标、课外目标

每晚看半小时书(可以是英文,小说,励志等)
