gerrit with git

  • 安装git工具
  • 配置git
  • 生成SSH Keys
  • 向Gerrit添加SSH keys
  • Git于Gerrit配合使用

###一、 安装git工具

1.Linux 
$ sudo yum install git 或者 $ sudo apt-get install git
2.Windows
Git官网下载.exe 文件安装即可

二、配置git

1.查看配置变量
$ git config -l
core.symlinks=false
core.autocrlf=true
core.fscache=true
color.diff=auto
color.status=auto
color.branch=auto
color.interactive=true
help.format=html
rebase.autosquash=true
http.sslcainfo=E:/Development tool/Git/git/Git/mingw64/ssl/certs/ca-bundle.crt
credential.helper=manager
filter.lfs.required=true
filter.lfs.clean=git-lfs clean -- %f
filter.lfs.smudge=git-lfs smudge -- %f
filter.lfs.process=git-lfs filter-process
user.name=daxiongit
user.email=daxiongit@qq.com
usr.email=daxiongit@qq.com
core.editor=emacs
core.autocrlf=false

2.配置用户名和邮箱
$ git config --global user.name "daxiongit"
$ git config --global user.email "daxiongit@qq.com"

三、生成 SSH keys

1.SSH keys
SSH 表示Secure Shell 或有时用于从远程计算机安全访问网络服务的Secure Socket Shell 协议，可以设置SSH密钥用来在计算机和Gerrit之间提供可靠的安全连接。
2.SSH 密钥文件位置
$ ls ~/.ssh/
id_rsa  id_rsa.pub
### id_rsa: 私有密钥
### id_rsa.pub: 公有密钥
3.生成新的SSH密钥
$ ssh-keygen -t rsa -C "daxiongit@qq.com"

四、向Gerrit 添加SSH密钥

1.复制id_rsa.pub 内容并复制下来

2.注册Gerrit 账户

[https://wikitech.wikimedia.org/wiki/Special:CreateAccount]

3.登录Gerrit

[http://gerrit.wikimedia.org/r/login/%23%2Fq%2Fstatus%3Aopen]

4.添加 SSH Public Keys

接着在右上角，点击您的用户名，然后选择设置(Settings)选项。在这里，我们创建了一个名为Daxiongit的帐户，以用来演示Gerrit的使用和演示。

###五、Gerrit配合Git使用

1.打开Git Bash获取 ssh-agent
$ eval 'ssh-agent'
2.将ssh密钥添加到ssh-agent
$ ssh-add ~/.ssh/id_rsa
！！！此时出现错误：
$ ssh-add ~/.ssh/id_rsa
Could not open a connection to your authentication agent.
！！！解决方案：
$ ssh-agent bash
$ ssh-add ~/.ssh/id_rsa

参考：【https://www.yiibai.com/gerrit/gerrit_overview.html】
