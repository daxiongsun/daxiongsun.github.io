###一、 表操作

1.创建数据库
  CREATE DATABASE mydb CHARACTER SET utf8;
2.创建数据表
  CREATE TABLE IF NOT EXISTS person(
   person_id SMALLINT UNSIGNED,
   fname VARCHAR(30),
   lname VARCHAR(20),
   genter ENUM('M','f'),
   CONSTRAINT pk_person PRIMARY KEY(person_id));

  CREATE TABLE IF NOT EXISTS favorate_food(
   person_id SMALLINT UNSIGNED,
   food VARCHAR(30),
   CONSTRAINT pk_per_food PRIMARY KEY(person_id,food),
   CONSTRAINT fk_food_fav_per FOREIGN KEY (person_id) REFERENCES person(person_id));

   例如：

   mysql> CREATE TABLE IF NOT EXISTS transaction(

二、约束(constraint)

1.主键约束(PRIMARY KEY)
  CONSTRAINT pk_person PRIMARY KEY(person_id)
2.外键约束(FOREIGN KEY(...) REFERENCES table(...))
  CONSTRAINT fk_fav_per FOREIGN KEY(person_id) REFERENCES person(person_id);
3.检查约束之枚举(ENUM)
  genter ENUM('M','F')

Angular基础一

一些基础用法

Quick Start

@Component

@Component({
    selector: 'app-hello',   // 用于模版中的标签
    templateUrl: './hello.component.html',  // 模版
    styleUrls: ['./hello.component.css'] // 模版样式
})

*ngFor

*ngFor : 在一个列表上进行迭代，为列表的每一个条目生成一个新的标签。
<li *ngFor="let name of names">{{name}}</li>

@Input

@Input 可以将数据传输给自组件，能够接受输入。
@Input() name: string;
<app-hello *ngFor="let myname of names" [name] = myname></app-hello>

@NgModule

在创建组件时必须把组件放入@NgModule的declarations中。
@NgModule({
declarations: [
  AppComponent,
  HelloComponent
]    
})
