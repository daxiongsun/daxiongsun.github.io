重要词汇 同步和异步 123456789101112131415161.同步（synchronous）异步（asynchronous）2.同步指一个进程在执行某个请求的时候，会等待该请求完成才会去执行其它任务，在这个等待期间，不做其它事情。异步指进程不需要一直等待下去，而是在等待的同时继续执行下面的操作。3.区别：同步需要等待，异步不需要等待同步：提交请求->等待服务器处理->处理完毕返回(等待期间不做其他事情)异步：请求通过事件出发->服务器处理->处理完毕(可同时处理其他任务)同步按顺序执行，异步彼此独立。异步和多线程不是同等关系，异步是最终结果，多线程只是实现异步的一种方式而已，实现异步可以采用多线程或者交给另外的进程来处理。4.优缺点同步可以避免出现死锁，读脏数据的发生。共享某一资源时可以使用同步。异步可以提高效率，但是安全性低。 并发和并行 123
